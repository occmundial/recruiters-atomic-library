# Welcome to recruiters-atomic-library

This Storybook page will allow you to interact with **headers** needed for occ's applications. Just click the **component** you would like to interact with 
and you're ready to go!

The **Canvas** tab will show the different props that a header needs, the data type for each prop and a 'Control' column which will allow you to interact with 
that specific prop, as well as the ability to change the viewport for that component.

On the other hand, the **Docs** tab will show the purpose of each prop, the default value for that prop and also the ability to interact with it.

**VERY IMPORTANT NOTE:** We recommend using the **canvas** tab to visualize the different use cases for a component since components don't work that well on the 
**docs** tab. The **docs** tab should only be used to learn what each prop does for the component and its default value.

**Github**: [**recruiters-atomic-library**](https://github.com/occmundial/recruiters-atomic-library/)
